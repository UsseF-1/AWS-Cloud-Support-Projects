# AWS Cloud Support Projects

This repository contains hands-on AWS projects focused specifically on Cloud Support Engineer roles.

The projects simulate real-world customer issues and troubleshooting scenarios involving AWS infrastructure, networking, and application connectivity.

---

## Focus Areas
- EC2 troubleshooting
- Security Groups & networking issues
- Connectivity failures (ICMP, HTTP, SSH)
- AWS service access problems
- Root cause analysis
- Ticket-style problem resolution
